# Glossary

**Adaptive:** Changes according to prior data using deterministic rules.

**Arousal threshold:** User-specific level at which the app asks the user to pause.

**Early pause:** A pause initiated before the user reports being almost unable to stop.

**Expert system:** Software that applies predefined rules and decision tables rather than a machine-learning model.

**Guided session:** Structured start–stop practice with timers, prompts, and haptics.

**Health hold:** Program state that disables guided sexual training after a red flag.

**Independence level:** Degree to which prompts and timers are reduced.

**Private session:** Optional light mode for a user who chooses to masturbate without a complete training protocol.

**Recovery:** Period after a pause during which stimulation stops and arousal is allowed to decrease.

**Rule reason code:** Stable identifier explaining why a recommendation was made.

**Urge surfing:** Brief practice of observing an urge without immediately acting on it.
