# Information Architecture

## Primary navigation

Use a five-tab structure:

1. **Today**
2. **Train**
3. **Progress**
4. **Learn**
5. **Settings**

## Today

- Current phase and week
- One primary recommended action
- “I’m aroused now” quick action
- Secondary recovery/health action
- Mini progress ring
- Next scheduled item

## Train

- Guided control session
- Urge surfing
- Breathing and relaxation
- Cardio plan
- Strength plan
- Recovery mobility

## Progress

- Weekly summary
- Awareness trend
- Successful pause trend
- Recovery trend
- Anxiety trend
- Exercise completion
- Explanations generated from deterministic templates

## Learn

- Body basics
- Control skills
- Anxiety and habits
- Relationships
- Exercise and sleep
- Medical warning signs

## Settings

- Privacy lock
- Discreet terminology
- Notification windows
- Haptics and sound
- HealthKit permission
- Data export/delete
- Medical disclaimer
- About the rule engine
