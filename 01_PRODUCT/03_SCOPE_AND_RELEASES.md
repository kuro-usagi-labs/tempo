# Scope and Releases

## MVP — 4-week local trainer

- Adult onboarding
- Baseline assessment
- Safety screening
- Daily plan
- Urge mode
- Guided start–stop mode
- Breathing orb
- Walking/jogging and bodyweight tasks
- Rule-based adaptation
- Local progress
- Neutral reminders
- Biometric lock
- Delete/export data
- 20 short educational modules

## Version 1.1

- Full 12-week progression
- Optional HealthKit activity read
- Better schedule conflict resolution
- Audio guidance packs
- Home Screen widgets with discreet labels
- App Intents shortcut: “Start quick check-in”
- Optional Apple Watch companion for haptics and heart-rate display

## Version 1.2

- Partner transfer phase
- Clinician-readable local PDF report
- Localization
- Alternative accessibility themes
- Advanced local trend explanations

## Explicitly deferred

- Cloud sync
- Subscription billing
- AI coach
- social/community features
- remote content management

Deferring these keeps privacy, testing, and product clarity strong.
