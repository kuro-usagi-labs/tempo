# Screen Specifications

## 1. Launch privacy gate

- Neutral logo animation.
- Face ID prompt.
- No sensitive text before unlock.

## 2. Today

Hero card:

- current phase;
- today’s primary action;
- estimated time;
- reason line;
- large CTA.

Below:

- **I’m aroused now** quick action;
- progress snapshot;
- next movement task;
- recovery status.

## 3. Urge check-in

Full-screen four-step flow with one question at a time. Progress shown as four small dots, not a long form.

## 4. Recommendation result

- Large icon or abstract symbol.
- Recommendation title.
- One-sentence reason.
- Primary action.
- Secondary “why this?” disclosure.
- Override only for non-safety decisions.

## 5. Guided session

- Immersive dark background.
- Large arousal dial.
- Cycle count.
- Hidden duration by default after Phase 2.
- Bottom action row for stable/rising/almost too late.
- Pause state transforms the entire screen.

## 6. Recovery screen

- Breathing orb.
- Countdown.
- “Rate again” control.
- No resume button until minimum pause time passes.

## 7. Progress

- Weekly summary sentence.
- Four metric cards.
- One chart at a time.
- Explanatory text below every chart.
- No population percentile.

## 8. Exercise

- Today’s workout steps.
- Large completion button.
- Replace/simplify action.
- HealthKit status if enabled.

## 9. Learn

- Short modules grouped by current need.
- “Why you’re seeing this” label.

## 10. Health check

- Clear symptom list.
- Stop-training notice.
- Guidance on clinician type.
- Emergency warning when appropriate.
