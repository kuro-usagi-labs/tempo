# “I’m Aroused Now” Mode

## Purpose

Give the user an immediate structured response without automatically converting every urge into masturbation.

## Entry point

Large primary home card:

> **I’m aroused now**  
> Get a private recommendation in 15 seconds.

In discreet mode:

> **Quick check-in**

## Four-question flow

1. **Intensity:** 1–10 slider.
2. **Context:** sexual desire, boredom, stress, loneliness, difficulty sleeping.
3. **Goal:** calm the urge, use it for training, private session.
4. **Safety:** pain, injury, urinary burning, unusual discharge, or none.

## Routing outcomes

### A. Urge surf

Used when the urge is linked to stress/boredom, training is too recent, or anxiety is high.

Five-minute experience:

- slow breathing orb;
- label the sensation;
- hands-off pause;
- optional standing/walking prompt;
- re-rate intensity.

### B. Guided control session

Used when:

- no safety flag;
- sufficient recovery time;
- within weekly session limit;
- user intent permits;
- current plan supports training.

### C. Private session

A neutral, lighter mode with safety reminders and optional logging. It does not produce a high score or encourage duration chasing.

### D. Recovery

Used after recent training, irritation, or exhaustion.

### E. Health check

Used on any red flag.

## User autonomy

The system may recommend rest, but a competent adult can override non-safety recommendations. Safety blocks cannot be bypassed inside guided mode. The user may close the app at any time.
