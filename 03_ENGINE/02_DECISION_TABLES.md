# Decision Tables

## Safety gate

| Condition | Result | Training blocked |
|---|---|---|
| Blood, fever, severe pain, acute injury | Urgent health guidance | Yes |
| Urinary burning or unusual discharge | Medical assessment guidance | Yes |
| Persistent pelvic/testicular pain | Medical assessment guidance | Yes |
| Mild friction irritation | Recovery 48–72 h and re-check | Yes |
| No symptoms | Continue evaluation | No |

## Session adaptation

| Recent result | Next-session change |
|---|---|
| Ejaculated before first pause twice | Lower pause threshold by 1; target 2 cycles |
| Late-stop rate >50% | Add stronger level-6 warning |
| 3+ cycles successful in 3 sessions | Add one cycle, maximum 5 |
| Recovery >120 sec repeatedly | Keep difficulty; extend guided breathing |
| Anxiety rising for 3 sessions | Reduce difficulty and hide duration |
| Irritation reported | Suspend sessions and start recovery plan |
| User repeatedly ignores rest | Display overtraining education; cap scheduled sessions |

## Exercise adaptation

| Condition | Change |
|---|---|
| Beginner completes 2 weeks comfortably | Add 5 minutes to one cardio session |
| Strength soreness >48 h | Reduce sets or replace with mobility |
| Missed 3 tasks | Reduce weekly load by 20%, do not stack tasks |
| Sleep <5 h | Recommend recovery or light walk |
| Injury reported | Disable affected exercise and show care guidance |
