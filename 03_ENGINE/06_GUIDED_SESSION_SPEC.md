# Guided Start–Stop Session Specification

## Session states

```text
PRECHECK
→ PREPARE
→ ACTIVE_LOW
→ ACTIVE_RISING
→ WARNING
→ PAUSED_RECOVERY
→ RESUME_READY
→ COMPLETED
→ POST_CHECK
```

Alternate terminal states:

```text
EARLY_COMPLETION
USER_CANCELLED
SAFETY_ABORT
TIME_LIMIT_REACHED
```

## Precheck

Ask:

- pain or irritation;
- anxiety 1–10;
- current arousal 1–10;
- enough private time;
- willingness to follow stop prompts.

## Prepare — 90 to 120 seconds

- breathing orb;
- relax jaw, abdomen, thighs, buttocks;
- choose sound/haptics;
- remind user not to chase duration.

## Active controls

- arousal dial 1–10;
- **Rising** button;
- **Stable** button;
- **Almost too late** button;
- **Pause now** button;
- **End session** button.

## Threshold behavior

### Level 1–5

Soft blue/indigo pulse. Minimal haptic.

### Level 6

Amber edge glow, single gentle haptic, copy:

> Slow down. Let your body soften.

### Level 7 or adaptive threshold

Red warning wave, stronger haptic, stop instruction:

> Stop now. Hands off. Breathe.

### “Almost too late”

Immediate emergency pause presentation. No punitive language.

## Recovery

- countdown range 30–120 seconds;
- animated breathing orb;
- user periodically re-rates arousal;
- resume only after reaching target level or choosing to end.

## Completion

The session can end after target cycles, time limit, or user choice. Erection loss is not failure and does not require compensatory stimulation.

## Maximums

- Default session cap: 20 minutes.
- Absolute guided cap: 25 minutes.
- Maximum cycles: 5.
- No automatic “try again” after early ejaculation.

## Post-check

- cycles completed;
- late-stop event;
- tension level;
- irritation/pain;
- anxiety after session;
- outcome feeling;
- optional note.
