## Milestone 6 — Progress and adaptation

- score calculations

## Milestone 7 — Privacy and hardening

- biometric lock
- encrypted notes/export
- deletion
- accessibility
- performance
- release build audit

## Milestone 8 — Optional integrations

- HealthKit
- widgets
- App Intents
- Apple Watch haptics

## Recommended build order

Do not begin with elaborate charts. Build the safety gate, rule engine, scheduler, and guided-session state machine first. The satisfying UI should sit on a correct deterministic core.
