# Definition of Done

A feature is done only when:

- product requirements are met;
- domain logic has unit tests;
- safety behavior is reviewed;
- UI supports dark mode and Dynamic Type;
- VoiceOver labels exist;
- Reduce Motion behavior exists;
- no sensitive logging is introduced;
- offline operation is verified;
- copy is reviewed for shame and unsupported claims;
- screenshots do not expose real personal data;
- the feature works after app background/foreground transitions;
- documentation is updated.

The MVP is done only when all core journeys work in airplane mode on a physical iPhone.
